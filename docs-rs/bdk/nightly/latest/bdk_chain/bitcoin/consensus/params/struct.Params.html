ยงmax_attainable_target: Target

The maximum attainable target value for these params.

Not all target values are attainable because consensus code uses the compact format to represent targets (see [CompactTarget]).

Note that this value differs from Bitcoin Coreโs powLimit field in that this value is attainable, but Bitcoin Coreโs is not. Specifically, because targets in Bitcoin are always rounded to the nearest float expressible in โcompact formโ, not all targets are attainable. Still, this should not affect consensus as the only place where the non-compact form of this is used in Bitcoin Coreโs consensus algorithm is in comparison and there are no compact-expressible values between Bitcoin Coreโs and the limit expressed here.
